# W0028 — a proc cannot call another proc

In the effect model the non-foreign world has three kinds — a **`func`** builds effects as values, a
**`system`/`map (Q) eff`** runs them, and a **`map`** is the data-parallel kernel — and `proc` is reserved for the
foreign/primitive boundary (`#foreign` / `@syscall` / `@intrinsic` / `@drop`). The few `proc`s that still
have a *body* (a `@drop` destructor, or a callable kept under `@allow(proc_not_primitive)`) are flat effect
**leaves**: W0028 keeps them from calling another `proc`.

The point is to keep the effect graph **depth-1**. If procs could call procs, effects would nest into a
call stack — exactly the depth that makes effects hard to read, test, and reason about. Anything you would
want to *share* between effect leaves is factored into a **`func` that builds an `Eff` value**, and a
`system` runs it. Reuse lives entirely in the value world, so a leaf never *needs* to call another leaf.

Permitted callees inside such a proc body:

- **`#foreign` proc / `@syscall` / `@intrinsic`** — the genuine primitive, the leaf an `Eff` wraps.
- **`func`** — pure value code, including a func that builds an `Eff`.
- **`map`** — a data-parallel kernel.

(In ordinary code you will not hit W0028 at all: a non-foreign `proc` is already an error, W0030. W0028
remains for the bodied exceptions above.)

## Severity

W0028 is a **warning by default**, tunable with:

- `--proc-leaf=warn` — the default (report, don't fail).
- `--proc-leaf=error` — fail the build.
- `--proc-leaf=allow` — silence it entirely.

Or suppress a single decl with `@allow(proc_calls_proc)`.

## Example

```arche
file :: opaque;
#foreign {
  @drop(file)
  fclose :: proc(own f: file);   // a primitive — fine to call from a leaf
}

// A @drop hook is a bodied proc (a leaf): it may run primitives, never another proc.
// BAD inside such a hook:  log_close(f);   // calling another non-foreign proc → W0028
// GOOD: run the primitive directly, or have a func build the Eff and a system run it.
```

See `docs/design/the-flat-effect-model.md` and `W0030` (proc is foreign-only).
