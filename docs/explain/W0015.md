# W0015 — unused enum

An `enum` type was declared but is never referenced anywhere (no variant access,
no value of the type, no mention in a signature). Usually dead code.

Same visibility model as [W0013](W0013.md): a **public** module enum is API and
is kept; a **private** enum referenced nowhere is dead; dependencies
(stdlib/core) are never linted; a **library** entry unit (no `main`) flags
nothing.

## Example

```arche
Unused :: enum { A, B }   // W0015: never referenced

main :: proc() {
  fmt.print("hi");
}
```

## How to silence

- **Use it** (e.g. `Unused.A`) — or delete it.
- **Prefix with `_`**: `_Draft :: enum { ... }`.
- **`@allow(unused_enum)`** (or `@allow(dead_code)`) on the decl.

## Not flagged

- Enums referenced anywhere — `E.variant`, a typed value, or a signature.
- Public module enums, dependency (stdlib/core) enums, library-entry enums.
- Names starting with `_`.
