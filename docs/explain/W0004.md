# W0004 — unused local

A local binding was declared but never read. Often that's a typo or
leftover from refactoring; rarely it's an intentional placeholder.

## Example

```arche
entry :: system {
  count := 5;   // W0004: never read
  fmt.printf("0\n");
}
#run entry
```

## How to silence

- **Read it.** Most cases are real — remove the binding or use it.
- **Prefix with `_`** if the value is intentionally unused but you want a
  name for documentation: `_scratch := compute_something();`.
- **`@allow(unused_local)`** on the enclosing decl silences the lint for
  the whole body (useful during early prototyping):
  ```arche
  @allow(unused_local)
  draft :: system {
    scratch := 5;  // not lint-warned here
  }
  ```

## Not flagged

- Function parameters (see W0005 once it ships).
- Opaque locals — the must-consume check (E0020) catches those instead.
- Names starting with `_`.
